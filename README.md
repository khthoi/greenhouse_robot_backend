## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Setup dự án, vào terminal của dự án và chạy lệnh:

```bash
$ npm install
```

## Hướng dẫn cách chạy project backend của dự án greenhouse_robot:

```bash
Bước 1: Thêm cơ sở dữ liệu với tên greenhouse_robot vào database

Bước 2:
Thêm file .env vào folder của dự án với nội dung:
DB_HOST=localhost
DB_PORT=3306 || your db port
DB_USERNAME=your_db_username
DB_PASSWORD=your_db_password
DB_NAME=greenhouse_robot
CORS_WHITELIST=http://localhost:3000,http://localhost:3001,http://127.0.0.1:3000

```

```bash
Bước 3: Chạy lệnh: 
# development
$ npm run start

```
